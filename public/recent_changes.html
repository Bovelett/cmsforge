CMSMS Forge is an evolving project that continues to get better as we go. Here are some of the recent changes:

2010-09-03
  • Added ability for project admins to delete projects.
  • Projects can now use github as their code repository.
2010-06-27
  • Removed the project comments. They proved to be less useful than anticipated. They may come back in a better form later on.
2010-02-13
  • Added last commit date and last release date to project summary
2010-02-13
  • Added full list of registered projects and file releases
  • Added RSS feeds for registered projects, file releases and individual projects
  • Added first letter filter to the project list
