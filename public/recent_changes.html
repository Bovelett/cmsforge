CMSMS Forge is an evolving project that continues to get better as we go. Here are some of the recent changes:

2010-02-13
  • Added full list of registered projects and file releases
  • Added RSS feeds for registered projects, file releases and individual projects
  • Added first letter filter to the project list
2009-04-12
  • Added open/closed state to the bug and FR lists
2009-04-07
  • Changed search backend. Allows for more fields to search (tags!), is more stable, and will make it easier to implement other types of searches down the road as well.
  • Added password recovery procedure
2009-04-05
  • Ability to request to join a project directly from the project's page
  • Fixed several reasons for multiple emails being sent from bugs and feature requests
  • Fixed problem which allows all project administrators to be locked out of a project
